# 🤖 Task Assistant - Gemini Clone

**Task Assistant** ek advanced Flutter application hai jo Google ke Gemini AI (Generative AI) ka upyog karke aapke rozana ke kaamo mein madad karta hai. Ise khas taur par **Ayush Online Center** ke liye design kiya gaya hai taaki daily tasks aur content creation aasaan ho sake.

---

## ✨ Key Features

* **AI Chat Interface**: Gemini AI ke saath fast aur intelligent baatcheet.
* **Generate Lyrics**: Ek click mein kisi bhi song ke liye creative Hindi/Bhojpuri lyrics generate karein.
* **Daily Reminders**: Rozana shaam 4:00 baje YouTube upload ke liye automatic notification (Local Timezone supported).
* **Custom Master Prompt**: App ki personality ko apne hisaab se set karne ki suvidha (Settings se configure karein).
* **Secure API Storage**: Aapki API key device par hi surakshit rehti hai.
* **Modern Dark UI**: Aankhon ko sukoon dene wala aur user-friendly Dark Mode interface.

---

## 🚀 Getting Started

### Prerequisites
* Flutter SDK (v3.0.0 ya usse upar)
* Gemini API Key ([Google AI Studio](https://aistudio.google.com/) se bilkul free me prapt karein)

### Installation
1.  Repository ko clone karein:
    ```bash
    git clone [https://github.com/businessak2510/task-assistant.git](https://github.com/businessak2510/task-assistant.git)
    ```
2.  Project folder mein jayein:
    ```bash
    cd task-assistant
    ```
3.  Dependencies install karein:
    ```bash
    flutter pub get
    ```
4.  App run karein:
    ```bash
    flutter run
    ```

---

## ⚙️ Configuration (App Setup)

1.  App kholne ke baad, left side se **Menu (Drawer)** open karein.
2.  Apni **Gemini API Key** wahan paste karein.
3.  **Master Prompt** set karein (Udahaaran: *"You are a professional lyricist for Ayush Online Center..."*).
4.  **'Save Configuration'** button par click karein aur apna assistant use karna shuru karein!
